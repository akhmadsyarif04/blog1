Pertama kita bikin dulu file connect.php. Dan jangan lupa jalankan local server kalian terlebih dulu bisa XAMPP atau Oracle. masukan skrip dibawah ini kedalam file connect.php tadi : 
    <?php
    	mysql_connect("localhost","root"," ");
    	mysql_select_db("nama_database");
    ?>

keterangan :
  • mysql_connect -> ini adalah perintah sql untuk mengubungkan kedatabase kalian
  • mysql_select_db -> ini digunakan untuk memilih nama database kalian, pasti kan sesuai dengan nama database kalian
  • "localhost" -> ini adalah nama host dari database kalian. Secara default nama nya localhost. Dan biasanya kalau misalnya tidak berhasil dengan localhost bisa menggunakan opsi lain yaitu 127.0.0.1
  • "root" -> ini adalah user dari database kalian secara default dia root.
  • " " -> ini adalah password. " " itu artinya password kosong. secara default passwordnya adalah kosong atau bisa juga "root"
Kalau sudah save dan letakkan didalam folder htdoc. misalkan dipartisi C: -> XAMPP -> htdocs.
Untuk cek apakah koneksi berhasil atau tidak maka gunakan skrip dibawah ini : 
    <?php
    $connect = mysql_connect("localhost","root"," ");
    $db = mysql_select_db("nama_database");

    if ($connect && $db ){
      die("koneksi berhasil");
    }else{
      die("gagal koneksi".mysql_connect_error);
    }
    ?>
Yaitu menggunakan if pada php. Dimana jikan $connect dan $db benar makan dia memberitahukan koneksi berhasil dan jika tidak makan gagal koneksi dengan pesan mysql_connect_error dari sql itu sendiri.
