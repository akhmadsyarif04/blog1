Kali ini saya akan berbagi trik animasi atau pun mengubah benduk border dengan CSS3.
membuat lingkaran : 
  <style>
  .contoh1{
    width:200px;
    height: 200px;
    background-color: #20cf46;
    border-radius: 110px 110px;
  }
  </style>
  <div class="contoh1"></div>

Membuat border bergerak dari kiri. untuk menampilkan klik Demo

contoh skrip : 

    <style>
      .contoh2{
        width:200px;
        height: 200px;
        background-color: #20cf46;
        animation: contoh2  5s;
        -webkit-animation: contoh2  5s ;
        -moz-animation: contoh2  5s;
      }
      @keyframes contoh2 {
      from{transform:translate(-1000px,0px);opacity:0;}
      to{transform:translate(0px,0px);opacity:1.1;}
      }
      @-webkit-keyframes contoh2 {
      from{transform:translate(-1000px,0px);opacity:0;}
      to{transform:translate(0px,0px);opacity:1.1;}
      }
      @-moz-keyframes contoh2 {
      from{transform:translate(-1000px,0px);opacity:0;}
      to{transform:translate(0px,0px);opacity:1.1;}
      }
      @-ms-keyframes contoh2 {
      from{transform:translate(-1000px,0px);opacity:0;}
      to{transform:translate(0px,0px);opacity:1.1;}
      }
      @-o-keyframes contoh2 {
      from{transform:translate(-1000px,0px);opacity:0;}
      to{transform:translate(0px,0px);opacity:1.1;}
      }
    </style>
    <div class="contoh2"></div>
Karena tidak semua browser mendukung CSS3 maka kita harus melakukan secara manual untuk membuat animasi nya bekerja pada browser lain

Keterangan :

-webkit-animation => untuk Crome & Safari.

-moz-animation => untuk Firefox.

-o-animation => untuk Opera.

-ms-animation => untuk Internet Explorer.
